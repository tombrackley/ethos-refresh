Control

The admin / leader surface — a cross-business performance lens covering health, matters, compliance, firm operating metrics, and team capacity.

Page structure

Health Snapshot Composite rating + four contributing signals
Active Matters Live matters, ordered by urgency
Compliance Obligation areas with overdue / in progress / up-to-date status
Firm Health Operating metrics with target deltas
Team Utilisation Per-person capacity

Widgets

Health Snapshot

Data
Composite org health rating, plus four contributing signals — Compliance, Risk Rating, Audit Readiness, Revenue. Each contributor shows a current value and a delta since the leader's last view.
Solves
"At a glance, how is the org doing — and which areas are moving the score up or down?"
Actions
Read only.

Active Matters

Data
Live matters with each matter's current status (Review / Active / On Hold). Sorted by urgency — Review first, On Hold last.
Solves
"Which matters need leadership attention right now?"
Actions
Click a matter to open its detail. Header chevron opens the full matters list.

Compliance

Data
Coarse-grained obligation areas (AML/KYC, CPD, Data Protection, Risk Assessments, Conflicts) with a status per area — overdue, in progress, or up to date.
Solves
"What compliance is overdue or in motion — and what does done look like?"
Actions
Hover a row for the precise status (e.g. "3 overdue"). Click the expand button to open the Compliance Checklist drawer — a per-area task list to review or mark done without leaving Control.

Firm Health

Data
Operating performance metrics — utilization, realization, collection, client NPS — each with current value, target, and delta since last period.
Solves
"Are firm operating metrics on target — which lines are moving against us?"
Actions
Read only.

Team Utilisation

Data
Each team member's current workload as a percentage of full capacity, surfaced as a horizontal bar chart.
Solves
"Who's stretched, who's got headroom — how evenly is the work distributed?"
Actions
Hover a row to see the person's active matter count. Sort by most or least utilised via the header dropdown.
