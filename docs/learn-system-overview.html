Key Principles

  • Learning Journeys are curated guides, not qualification pathways. The org creates them; Ethika/Ethos does not certify or qualify them.
  • CPD is a separate compliance system — hours are mandated by regulation, not by the org's learning journeys.
  • Skills Profile is the overarching competency framework. Journeys may help develop some skills, and some skills map to CPD requirements, but they are independent systems.
  • Log Learning is a reflection, not a completion. Users share what they learned; admins acknowledge or respond with feedback.
  • Items in a journey are non-sequential — there is no prescribed order or dependency.
